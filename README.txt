Wild Rift Guide｜目前已整理裝備更新包

內容：
- 物理中階：18 件
- 魔法中階：18 件
- 防禦中階：15 件
- 輔助中階：13 件
- 分類內保留重複裝備，不建立共用素材分類
- 點擊裝備可查看價格、屬性與目前已整理的效果
- 支援手機與電腦版

使用方式：
1. 解壓縮。
2. 先直接雙擊 index.html 預覽。
3. 將整個資料夾內容複製到 wild-rift-guide 專案根目錄。
4. 在 GitHub Desktop 輸入 Summary，例如：
   Update intermediate equipment database
5. 按 Commit to main。
6. 按 Push origin。

注意：
- 此包是依照本次對話目前整理完成的資料製作。
- 輔助分類依「全部為其他分類重複裝備」的規則呈現。
- 若要無損合併你電腦上目前的完整網站（符文、召喚師技能、鞋子及既有首頁），仍應用最新版專案 ZIP 進行合併。
